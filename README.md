Raspberry Pi Emulation Environment
----------------------------------

Raspberry Pi emulation environment for OS X.

Setup Instructions
-------------------

1. Run `rake install`
1. Run `rake boot`

This should work, but if it doesn't, use the sites referenced below to troubleshoot.
